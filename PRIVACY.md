# Политика конфиденциальности MKS Push

*Последнее обновление: 6 июня 2026 г.*

## Данные, которые собирает приложение

Приложение **MKS Push** собирает следующие данные исключительно для обеспечения своей работы:

| Данные | Назначение |
|---|---|
| **Push-токен** | Технический идентификатор, выдаваемый Apple / Google для доставки push-уведомлений на ваше устройство |
| **User ID** | Уникальный идентификатор, присваиваемый сервером при регистрации устройства |

## Как используются данные

- Push-токен и User ID отправляются на сервер **https://mkspush.ru** при регистрации устройства
- Эти данные используются **только** для маршрутизации push-уведомлений от подключённых вами веб-сервисов и IoT-устройств к вашему телефону
- Push-токен хранится на сервере до момента отключения устройства

## Что приложение НЕ делает

- **Не читает** содержимое ваших сообщений или уведомлений
- **Не собирает** геолокацию, контакты, фотографии или другие личные данные
- **Не передаёт** данные третьим лицам
- **Не использует** данные для рекламы или аналитики

## Хранение на устройстве

На устройстве сохраняется только **User ID** (в защищённом хранилище). Никакие другие данные не хранятся локально.

## Удаление данных

При нажатии кнопки «Отключить устройство» в приложении:
1. User ID удаляется с устройства
2. Серверу отправляется запрос на удаление сессии
3. Push-токен и User ID удаляются с сервера
